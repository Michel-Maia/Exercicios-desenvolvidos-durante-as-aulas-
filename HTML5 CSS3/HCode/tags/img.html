Aula tag - img

Tag usada para inserir uma imagem na página.
A tag não possui conteúdo
Os atributos src e alt são obrigatórios.

Atributos

alt

Define um texto alternativo para a imagem
- Texto

crossorigin

Define como o elemento lida com solicitações de origem cruzada
- anonumous, use-credentials

height

Define a altura da imagem
- número

ismap

Define a imagem como um mapa de imagem do lado do servidor
- ismap

longdesc

Define uma descrição detalhada da imagem
#idDaDescricao

sizes

Define tamanhos alternativos da imagem

src

Define qual a fonte/link da imagem
- URL

srcset

Define a URL da imagem em diferentes situações
- URL

usemap

Define a imagem como um mapa de imagem do lado do cliente
#nomeDoMapa

width

Define a largura da imagem
- número
