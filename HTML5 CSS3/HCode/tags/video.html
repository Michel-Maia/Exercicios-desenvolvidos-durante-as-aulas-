Aula tag - video

Tag usada para definir um vídeo, filme ou transmissão de vídeo.
Atualmente é suportado os formatos: MP4, WEBM OGG.
Qualquer texto no conteúdo da tag será exibido em navegadores que não suportam a tag video.

Atributos

autoplay

Começa a reproduzir assim que os dados do vídeo estiverem carregados.
- autoplay

controls

Habilita os controles de reprodução do vídeo.
- controls

height

Define a altura do player do vídeo.
- Número

loop

Define se o vídeo deve começar a reproduzir novamente após terminar - loop

muted

Define o volume do vídeo como mutado
- muted

poster

Define uma linguagem para ser exibida enquanto o vídeo estiver sendo carregado ou antes do usuário clicar em reproduzir
- URL

preload

Define como o vídeo deve ser carregado quando a página é carregada.
- auto, metadata, none

src

Define qual a fonte/link do vídeo
- URL

width

Define a largura do player do vídeo
- número
