Aula tag - object

Tag usada para definir um objeto incorporado no documento.
É possível incorporar um áudio, vídeo, applets Java, ActiveX, PDF e etc.
É possível para parâmetros para o objeto incorporado usando a tag. param Estilo CSS padrão

Atributos

data

Define a URL do recurso que será usado como objeto. URL

form

Define a que formulário esse objeto pertence. ID do formulário

height

Define a altura do objeto. Número

name

Define o nome do objeto. Texto

type

Define o tipo de mídia do objeto. Tipo de mídia

usemap

Define o nome do mapa de imagem para seu usado com o objeto. ID do mapa

width

Define a largura do objeto. Número
