Aula tag - track

Tag usada para definir uma faixa de texto em uma tag de mídia
Normalmente usada para definir legendas em vídeos.

Atributos

default

Define se esta é a faixa selecionada por padrão
- default

kind

Define o tipo da faixa de texto
- captions, chapters, descriptions, metadata, subtitles

label

Define o rótulo da faixa de texto.
- NomeDoInput.dir

src

Define a URL da fonte da faixa de texto
- URL

srclang

Define qual a linguagem dos dados da faixa de texto. Obrigatório código da linguagem para legendas
