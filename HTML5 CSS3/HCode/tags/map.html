Aula tag - map

Tag usada para criar um mapa de imagem clicável.
O atributo name é obrigatório para relacionar com a tag img
A tag map defini as áreas clicáveis usando várias tags area.
name Define um nome para relacionar com o atributo usemap da tag ig. - Nome da tag

Atributos

Tag usada para definir áreas clicáveis em um mapa de imagem.
Esta tag deve ser sempre filha de uma tag map.
alt Define um texto alternativo para a imagem. - Texto
coords Define as coordenadas da área. - x1, y1, x2, y2 para shape rect | x, y, raio para shape circle | x1, y1, x2, y2,....,xn, yn para shape poly.
download Usado para forçar o download do destino do link. - Se for definido um texto no valor, o mesmo será usado como nome do arquivo que será baixado no download.
href Define qual o endereço do link de destino. - URL
hreflang Define qual a linguagen do link de destino. - Media Query rel Define qual a relação do documento atual e o de destino. - alternate, author, bookmark, external, help, license, next, nofollow, noreferrer, noopener, prev, search, tag
shape Define uma forma para a área. - defaut, rect, circle, poly
target Define onde o link deverá ser aberto. _blank, _parent, _self, _top, Nome de um Frame
type Define o tipo de mídia do link de destino. - Tipo de mídia (MIME type)
