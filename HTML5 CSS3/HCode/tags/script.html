Aula Tag - script

Tag usada para definir um script em Javascript executado no lado do cliente.
A tag permite o script ser escrito dentro dela ou apontar uma fonte externa. todos os atributos globais Estilo CSS padrão

Atributos exclusivos

async

Define que o script será executadp de forma assincrona. (Funciona async apenas para scripts externos)

charset

Define o padrão de caracteres usado. Padrão de caracteres que será usado

defer

Define que o script só será executado depois da página ser carregada. (defer)

src

Define a fonte externa do script. URL

type

Define o tipo de mídia do script. Tipo de mídia
