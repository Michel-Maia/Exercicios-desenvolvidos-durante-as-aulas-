Aula tag - audio

Tag usada para definir um som, música ou transmissões de áudio
Atualmente suporta os formatos: mp3, WAV e OGG.
Qualquer texto no conteúdo da tag será exibido em navegadores que não suportam a tag audio

Atributos

autoplay

Começa a reproduzir assim que os dados do áudio estiverem carregados.
- autoplay

controls

Habilita os controles de reprodução do áudio.
- controls

loop

Define se o áudio deve começar a reproduzir novamente após terminar - loop

muted

Define o volume do áudio como mutado
- muted

preload

Define como o áudio deve ser carregado quando a página é carregada.
- auto, metadata, none

src

Define qual a fonte/link do áudio
- URL
