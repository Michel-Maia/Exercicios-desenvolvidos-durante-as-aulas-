Aula tag - link

Tag usada para linkar arquivo externo, uso comum é para associar documento CSS ex link rel="stylesheet"

Atributos

crossorigin

Define como o elemento lida com solicitações de origem cruzada. anonymous, use-credentials

href

Define o link referente ao documento que será associado. URL

hreflang

Define a linguagem do documento associado. Código de linguagem

media

Define em dispositivo se aplica este documento. Consulta de mídia

rel

Define a o relacionamento entre o documento atual e o associado. alternante, author, dns-prefetch, help, icon, license, next, pingback, preconnect, prefetch, preload, prerender, prev, search, stylesheet

sizes

Define um tamanho de largura e altura para associação de ícones. AlturaXLargura

type

Define o tipo de mídia do documento associado. Tipo de mídia
