tag ol

Tag usada para definir uma lista ordenada.
A ordem pode ser numérica ou alfabética.

reversed Define a ordem da lista com decrescente. - reversed
start Define qual o valor de início da lista. - Número
type Define o tipo de marcação de ordem da lista. - 1, A, a , I, i
