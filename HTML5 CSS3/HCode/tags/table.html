Aula table

Tag usada para definir uma tabela.
Todos os atributos além dos globais fora removidos do HTML5.
Apenas a tag table não cria toda a tabela.
É necessário usar as tags tr e td para criar as linhas e células respectivamente.
Uma tabela mais complexa ainda pode conter as tags caption, col, colgroup, thhead, tfoot e tbody.
Historicamente as tabelas eram usadas para criar o layout da página porém esse seria um mal uso desta tag
Recebe os atributos globais

Tag tr

Tag usada para definir linhas dentro de uma tabela
Está tag deve conter um ou mais tags th e/ou td.
Todos os atributos além dos globais foram removidos no HTML 5.

Tag td

Tag usada para criar uma célula da tabela.
A tag deve ser usada dentro de uma tag tr.



Atributos


colspan Define o número de colunas que esta célula deve ocupar. - Número
headers Define uma mais células de cabeçalho relacionadas. - ID da célula
rowspan Define o número de linhas que esta célula deve ocupar. - URL

Tag thead

Tag usada para agrupar o conteúdo do cabeçalho da tabela.
A tag é usada em conjunto com as tags tbody e tfoot.
Os navegadores podem usar esta definição para ativar a rolagem do corpo da tabela.
A tag deve ser filha de uma tag table.
A tag pode vir após uma tag caption e/ou colgroup.
A tag deve vir antes das tags tbody e tfoot.
A tag deve ter um ou mais tags tr dentro dela.
A tag não afeta visualmente a tabela, mas pode ser estilizada com CSS.

Tag tbody

Tag usada para agrupar o conteúdo do corpo da tabela.
A tag é usada em conjunto com as tags thead e tfoot.
Os navegadores podem usar esta definição para ativar a rolagem do corpo da tabela.
A tag deve ser filha de uma tag table.
A tag deve vir depois da tag thead e antes da tag tfoot.
A tag não afeta visualmente a tabela, mas pode ser estilizada com CSS.

Tag tfoot

Tag usada para agrupar o conteúdo do rodapé da tabela.
A tag é usada em conjunto com as tags tbody e tfoot.
Os navegadores podem usar esta definição para ativar a rolagem do corpo da tabela.
A tag deve ser filha de uma tag table.
A tag deve vir depois da tag thead e tbody.
A tag não afeta visualmente a tabela, mas pode ser estilizada com CSS.

Tag th

Tag usada para definir uma célula de cabeçalho.
A tag deve ser usada dentro de uma tag tr.

abbr Define uma abreviação do conteúdo da célula do cabeçalho. - Texto.
colspan Define o número de colunas que esta célula deve ocupar. - Número
headers Define uma mais células de cabeçalho relacionadas
- ID da célula rowspan Define o número de linhas que esta célula deve ocupar. - URL
scope Define se a célula é um cabeçalho de linha, coluna, grupo de linha ou grupo de coluna. - col, colgroup, row, rowgroup sorted Define a direção da ordem da coluna. - reversed, number, reversed number, number reversed

Tag colgroup

Tag usada para definir propriedades em todas as células que formam uma coluna.
Esta tag deve ser usada dentro de uma tag colgroup

span Define o número de colunas que esse grupo deve abranger. - Número

Tag col

Tag usada para definir propriedades em todas as células que formam uma coluna.
Esta tag deve ser usada dentro de uma tag colgroup

span Define o número de colunas que esse grupo deve abranger. - Número

Tag caption

Tag usada para definir uma legenda da tabela.
A tabela deve ser inserida imediatamente após a abertura da tag table.
