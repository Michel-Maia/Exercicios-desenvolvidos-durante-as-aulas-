Aula tag - source

Tag usada para definir uma fonte de mídia.
Usada junto com as tags audio, video e picture.

Atributos

src

Define a fonte externa do source
- URL

srcset

Define uma fonte em situações diferentes
- URL

media

Define em que tipo de mídia essa fonte de dados se aplica
- Consulta de mídia

sizes

Define tamanhos diferentes

type

Define o tipo de mídia do arquivo
- tipo de mídia
