Aula tag - details

Tag usada para definir detalhes adicionais que o usuário pode exibir ou ocultar.
Qualquer tipo de conteúdo pode ser colocado dentro desta tag.
Por padrão o conteúdo dos detalhes não é exibido para o usuário.
A tag deve ter uma tag summary dentro dela.
open Define se os detalhes devem ficar vissíveis para o usuário. - open
