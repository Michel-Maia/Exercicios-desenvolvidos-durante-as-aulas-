tag bdi

  • Tag usada para definir parte de um texto que pode ser formada em uma direção diferente do padrão.
  • Essa tag é útil ao exibir conteúdo gerado pelo usuário com uma direção desconhecida

  5. 米歇尔

tag bdo

  • Tag usada para definir a direção do texto.
  • Essa tag substitui a direção do texto atual.
  • O atributo dir é obrigatório.

tag ruby

  • Tag usada para definir uma anotação em ruby.
  • Anotação ruby é um pequeno texto extra, anexado ao texto principal para indicar a pronúncia ou o significado dos caracteres correspondentes.
  • Esse tipo de anotação é frequentemente usado em publicações japonesas

tag rp

  • Tag usada para definir parênteses em navegadores que não suportam tags de anotação ruby

tag rt

  • Tag usada para definir o texto ruby de uma anotação em ruby.
  • A tag fornece informações de pronúncia, tradução ou transliteração do Leste Asiático.
  • A tag sempre deve ser usada em uma tag ruby.

Exemplo - 学生 ( Gakusei)
