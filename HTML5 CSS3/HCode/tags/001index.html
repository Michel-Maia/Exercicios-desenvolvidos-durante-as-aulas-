tag abbr

  • Tag usada para definir uma abreviação
  • Importante para fornecer informações úteis para navegadores, sistema de tradução e mecanismos de busca

Exemplo - Lorem ipsum dolor. SP Magnam, totam.

tag b

  • Tag usada para definir um texto em negrito mas sem nenhuma importância extra.
  • Antes de usar a tag considere o uso das tags:
    • h1 a h6;
    • em;
    • strong;
    • mark;

Exemplo - Lorem ipsum dolor, $$$ sit amet consectetur adipisicing elit.

tag bdi

  • Tag usada para definir parte de um texto que pode ser formada em uma direção diferente do padrão.
  • Essa tag é útil ao exibir conteúdo gerado pelo usuário com uma direção desconhecida

Exemplo - Lorem ipsum dolor, sit amet consectetur adipisicing elit.

  1. انزو
  2. ليليان
  3. Enzo
  4. Jéssica
  5. 米歇尔
  6. 玛雅

tag bdo

  • Tag usada para definir a direção do texto.
  • Essa tag substitui a direção do texto atual.
  • O atributo dir é obrigatório.

dir Define a direção do texto dentro do elemento. - ltr, rtl
