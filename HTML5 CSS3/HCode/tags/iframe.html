Aula tag - iframe

Tag usada para definir um quadro embutido.
O quadro embutido é usado para incorporar outro documento no documento HTML atual.

Atributos

height

- Define a altura do quadro
- número

width

- define a largura do quadro - texto

name

- define o nome do quadro
- URL

sandbox

- ativa comteúdos restritos por padrão
- allow-forms, allow-pointer-lock, allow-popups, allow-same-origin, allow-scripts, allow-top-navigation

src

- define qual a fonte do documento a ser embutido
- URL

srcdoc

- define qual a estrutura do documento a ser embutido
- HTML
