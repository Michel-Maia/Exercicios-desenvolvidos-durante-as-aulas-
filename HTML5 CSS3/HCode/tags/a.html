Aula tag - a

Tag usada para criar hyperlink e vincular uma página HTML com outra.
O atributo mais usado é o href.
Por padrão o conteúdo da tag é mostrado sublinhado.

Youtube

Atributos

download

Usado para forçar o download do destino do link
- Se for definido um texto no valor o mesmo será usado como nome do arquivo que será baixado no download.

href

Define qual endereço do link de destino
- URL

hreflag

Define qual a linguaguem do link de destino
- Código de linguagem

media

Define qual a mídia ou dispositivo do link de destino
- Media Query

ping

Endereço de rastreio para monitorar os cliques no link de destino.
- URL

referrerpolicy

Define qual referenciador enviar
- no-referrer, no-referrer-when-downgrade, origin, origin-when-cross-origin, unsafe-url

rel

Define qual a relação do documento atual e o de destino
- alternate, author, bookmark, external, help, license, next, nofollow, norerrer, noopener, prev, search, tag

target

Define onde o link deverá ser aberto
_blank, _parent, _self, _top, Nome de um frame

type

Define o tipo de mídia do link de destino - Tipo de mídia (MIME type)
