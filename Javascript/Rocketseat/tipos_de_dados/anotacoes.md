
# Tipos de dados

Conforme o ECMAScript standard temos  tipos de dados:

* Data types
	* Primitive / Primitive value
	* Structural 
	* Strutural Primitive 

## Primitivos (não sofre alterações)


* String
* Number
* Boolean
* undefined
* Symbol
* BigInt 


## Estruturais


*Object
	*Array
	* Map
	* Set
	* Date
*Function


## Primitivo Estrutural / Structural Root Primitive


* null
